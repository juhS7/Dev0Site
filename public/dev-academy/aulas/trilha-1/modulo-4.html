🎯 Objetivo

Ler e escrever Python com fluência o suficiente para automatizar tarefas, consumir APIs, processar dados e subir uma API REST com FastAPI — o stack mais comum quando o trabalho mistura backend e IA.

1. Por que Python no currículo de pleno?

Python é a língua franca de IA/ML, automação, scripts internos e ferramentas de dados. Mesmo um time 100% TypeScript no produto principal acaba escrevendo Python para pipelines de ETL, scripts de migração e protótipos de LLM. Saber ler Python e escrever bem o suficiente para entregar uma API é hoje uma competência mínima de pleno.

2. Sintaxe e tipos

Python usa indentação como sintaxe. Não há chaves. Cuidado com mistura de tabs e espaços (use 4 espaços, sempre).

nome: str = "Ana"
idade: int = 30
ativo: bool = True
tags: list[str] = ["py", "fastapi"]
user: dict[str, object] = {"nome": nome, "idade": idade}

if idade >= 18 and ativo:
    print(f"{nome} pode entrar")
else:
    print("bloqueado")
💡 Dica

Type hints (: str, : int) são opcionais em runtime, mas mypy / Pyright usam para checar o código estaticamente. Em código profissional eles são esperados.

3. Funções, lambdas e list comprehensions

def somar(a: int, b: int = 0) -> int:
    return a + b

dobrar = lambda x: x * 2

# list comprehension — substitui map/filter
numeros = [1, 2, 3, 4, 5]
pares_dobrados = [n * 2 for n in numeros if n % 2 == 0]
# [4, 8]

# dict comprehension
quadrados = {n: n*n for n in range(5)}
# {0:0, 1:1, 2:4, 3:9, 4:16}

4. Classes e OOP

from dataclasses import dataclass

@dataclass
class User:
    nome: str
    idade: int
    ativo: bool = True

    def saudacao(self) -> str:
        return f"Olá, {self.nome}"

u = User(nome="Bia", idade=25)
print(u.saudacao())

@dataclass elimina o boilerplate de __init__, __repr__ e __eq__. Em código novo, prefira dataclasses ou Pydantic models a classes "cruas".

5. Arquivos, JSON e pathlib

import json
from pathlib import Path

config_path = Path("config.json")
config = json.loads(config_path.read_text(encoding="utf-8"))

config["versao"] = "2.0"
config_path.write_text(json.dumps(config, indent=2, ensure_ascii=False))
⚠ Atenção

Esqueça open() + os.path.join() em código novo. pathlib.Path é orientado a objetos, multiplataforma e muito mais legível.

7. Sua primeira API REST com FastAPI

FastAPI usa type hints + Pydantic para gerar validação, serialização e documentação Swagger automaticamente. É o framework mais usado em backends Python modernos.

from fastapi import FastAPI, HTTPException
from pydantic import BaseModel

app = FastAPI(title="DevPulse API")

class Metric(BaseModel):
    name: str
    value: float
    tags: list[str] = []

DB: dict[str, Metric] = {}

@app.post("/metrics", status_code=201)
def create(m: Metric) -> Metric:
    DB[m.name] = m
    return m

@app.get("/metrics/{name}")
def get(name: str) -> Metric:
    if name not in DB:
        raise HTTPException(404, "métrica não encontrada")
    return DB[name]
